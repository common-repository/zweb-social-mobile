Tích hợp nút gọi trên mobile và desktop

== Description ==

Tích hợp nút gọi tương tác với khách hàng khi vào bằng điện thoại và máy tính

Ưu điểm:
*   Dễ dàng tương tác với khách hàng
*   Giao diện dễ sử dụng và tùy chỉnh
*   Chỉ cần 1 chạm là khách hàng có thể liên hệ với bạn bằng zalo, email, và hotline
*   Hỗ trợ nâng cấp plugins lên bản PRO tại https://zweb.vn

== Installation ==

1. Cài đặt Plugin như bình thường
2. Kích hoạt Plugin ZWeb - Social Mobile
3. Vào ZWeb - Social Mobile trong phần Cài Đặt thể thêm thông tin
